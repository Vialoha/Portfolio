WeHive

A solução definitiva para conectar utilizadores com ideias semelhantes, para que possam contribuir com as suas competências técnicas ou artísticas para um projeto. Com uma interface simples, os utilizadores podem navegar facilmente por uma vasta gama de listas de projetos públicos, encontrar aqueles que se alinham com os seus interesses e especialidades, e enviar mensagens públicas para oferecer colaboração. Diga olá às colaborações produtivas e apaixonadas que combinam os mundos criativo e tecnológico.


Desafios e Objetivos:

  • Ativar a funcionalidade CRUD para o perfil do utilizador, projetos e mensagens (Ao utilizar pedidos post, put, get e delete associados a rotas específicas);
  • Implementar mensagens/Caixa de Entrada (criar uma relação entre o esquema da mensagem e o esquema do utilizador);
  • Rotas de autorização (para confirmar que o utilizador está com sessão iniciada e possui um JWT - JSON Web Token válido);
  • Ativar projetos privados/públicos e apresentá-los de forma diferente na aplicação (criar um esquema de projeto com a propriedade privada com um valor por defeito igual a "público" e ativar no formulário a opção de o alterar para privado);
  • Implementar tags nos projetos, para serem utilizadas na barra de pesquisa (esquema do projeto com mais uma propriedade denominada tags).


Ferramentas Ulilizadas:

  • Biblioteca para o UI: React;
  • Arquitetura: Node.js;
  • CSS Framework: Bootstrap - para uma abordagem responsiva e com foco no "mobile first";
  • Armazenamento das Imagens(projetos e foto de perfil): Cloudinary;
  • Armazenamento do Back-end: MongoDB Atlas;
  • Testar rotas no VS Code: Rest API ThunderClient;
  • Lançamento front-end: Netlify;
  • Lançamento back-end: Cyclic;


Backlog - Futuras Funcionalidades:

  • Melhorar o sistema de mensagens de pessoa para pessoa em vez de público e ligá-lo a um projeto específico (relação);
  • Implementação de mais tags para projetos;
  • Mais opções para atualizar as definições do perfil do utilizador, como a atualização do campo de confirmação da palavra-passe e da fotografia de perfil;
  • Criação de informação de rodapé e página inicial;
  • Criação de um logótipo e melhoria da imagem da marca.
